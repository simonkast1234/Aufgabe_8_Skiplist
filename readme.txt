To run this solution, you have to add JUnit4 to the class path:
1. find the error: *.junit.* is red, message is "cannot resolve symbol junit"
2. ALT+Enter while the cursor is in the red junit field
3. Choose "add Junit4 to the classpath" and confirm (twice)

Alternative:
1. If you don't want to run the tests, you can just simply delete the *Test classes (BaseListTest and SkipListTest).
